# bhOS 생태계 — 종합 인덱스

> 이 문서는 다른 문서들을 연결하는 허브입니다. 상세 내용은 각 문서를 참조하세요. 2026-03-20

---

## 문서 맵

```
docs/dev/
├── VISION.md           ← 비전 + 핵심 전략 ("싸우지 않고 이긴다")
├── ARCHITECTURE.md     ← 시스템 아키텍처 (Claw Core, 이벤트 버스, Federated Memory)
├── PRODUCTS.md         ← 제품별 상세 (ClawTalk, ClawMemory, ClawMeeting, ClawNote, NanoClaw)
├── BUSINESS-MODEL.md   ← 수익 모델 + 빌드 순서
├── PRD.md              ← ClawTalk MVP 상세 스펙
├── DECISIONS.md        ← ADR 13개 (핵심 결정 기록)
├── JOURNAL.md          ← 개발 일지
├── TODO.md             ← 할 일 추적
├── HANDOFF.md          ← 세션 간 핸드오프
├── poc/
│   └── 001-schedule-agent.md  ← PoC 결과
└── ECOSYSTEM-VISION.md ← 이 파일 (인덱스)

docs/research/
├── RESEARCH-INDEX.md   ← 리서치 인덱스 (완료 + 추가 필요 목록)
├── 01-protocol-comparison.md  ← A2A/MCP/Matrix 비교
└── 05-messenger-agent-comparison.md ← 메신저별 에이전트 통신

docs/
├── AGENT-MESSENGER-ANALYSIS.md ← 전문가 패널 전략 분석
├── SECURITY-RISK-ANALYSIS.md   ← 보안 분석
└── TELEGRAM-IPV6-FIX.md        ← 트러블슈팅
```
