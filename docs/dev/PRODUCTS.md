## ClawTalk — 에이전트 네이티브 메신저

**역할**: 사람과 에이전트가 1:1, 1:다, 다:다로 소통하는 메신저
**시장**: 글로벌 메시징 $100B+ (장기)
**상세 PRD**: `docs/dev/PRD.md`

핵심 기능:
- Room (사람 N + 에이전트 M 혼합 대화방)
- Task Engine (A2A 기반 상태 머신 + HITL)
- Agent Card (에이전트 등록/발견)
- Telegram Bridge (첫 진입점 → 웹앱 유도)
- Calendar + Time Slot Manager
- 온보딩 (가상봇 체험 → 30초 아하 모먼트)
- BYOK + OpenRouter (무료 체험)

## ClawMemory — 기억 + 관계 + 맥락

**역할**: 노트가 아닌 **기억 시스템**. 사람, 관계, 대화, 맥락을 통합.
**시장**: 다른 제품의 가치를 10배 증폭하는 차별화 엔진

### 3개 계층

| 계층 | 내용 |
|------|------|
| **Knowledge** | 마크다운 노트 + 자동 온톨로지 추출(LLM) + 벡터 검색 |
| **Relationship** | 사람 엔티티 + 관계 유형(가족/친구/동료/거래처/고객) + 관계 온도 + 상호작용 히스토리 |
| **Context** | AI 에이전트 메모리 통합 + 세션 히스토리 + 시간축 |

### 온톨로지 전략

> "마크다운 퍼스트 + 사용자 명시 관계 정의 + 자동 온톨로지 보강 + 벡터 검색"

- 사용자: 마크다운 작성 + 사람/관계 명시 등록
- 시스템: 자동 엔티티 추출 + 임베딩 + 온톨로지 보강
- 온톨로지를 사용자에게 강제하지 않음

### 킬러 유즈케이스: 미팅 준비 자동화

```
캘린더 미팅 감지 → 참석자 관계 조회 → 과거 맥락 검색
→ 브리핑 자동 생성 → ClawTalk 전달 (1시간 전)
→ 미팅 녹음 (ClawMeeting) → 회의록 → 관계 업데이트 → 온도 조정
```

### 옵시디언 전환 전략

```
Phase 0: 옵시디언 = UI, CLI = 관계/인덱싱
Phase 1: 옵시디언 플러그인 (관계 시각화, 클라우드 동기화=유료)
Phase 2: ClawNote 자체 클라이언트 (옵시디언 데이터 호환)
```

## ClawMeeting — 미팅 녹음 + 전사 + 요약

**역할**: 한국어 특화 미팅 전사 서비스
**시장**: 미팅 전사 (Fireflies $100M+ ARR, 검증됨)
**전략**: 한국어 끝판왕 품질, Day 1 클라우드 서비스, 즉시 수익

핵심 기능:
- 실시간 음성 녹음 + 한국어 전사
- 화자 분리 (diarization)
- 실시간 문단 정리 (Tiro 스타일)
- AI 요약 + 액션 아이템 추출
- ClawMemory 연동 (전사본 → 관계 업데이트)
- 클라우드 저장 (유료)

---

## ClawNote — 마크다운 노트 UI

**역할**: ClawMemory의 프론트엔드
**전략**: 초기 옵시디언으로 대체, 점진적 자체 개발

초기: 옵시디언 볼트가 데이터 저장소
후기: 자체 에디터 (옵시디언 마크다운 호환 유지)
수익: 클라우드 동기화 (옵시디언 Sync 모델, 플러그인으로도 제공 가능)

## NanoClaw — 에이전트 런타임

**역할**: 모든 모듈의 기반. 에이전트가 실제 작업을 수행하는 엔진.
**전략**: 오픈소스. 직접 수익 없음. 생태계 성장 동력.

이미 운영 중:
- Docker 컨테이너 격리
- MCP (도구 접근)
- 멀티채널 (Telegram, Slack, Discord, WhatsApp)
- BYOK (Claude OAuth, API Key, Ollama)
